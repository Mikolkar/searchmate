## Installation

#### Installation with poetry:

```shell 
poetry install
```
#### Installation with pip:

```shell
pip install -e
```

## Usage

#### With poetry:

```shell
poetry run python -m searchmate
```

#### With pip:

```shell
python -m searchmate
```
